## 使い方

1. Suno → Create → **Custom** モードON
2. Title / Style / Lyrics に上記をコピペ
3. Persona欄: 教祖様の声Voice (RYOSEI-001等) が登録済みなら選択
4. Generate → 2バージョン生成、優しい方を採用
5. mp3 DL → プロジェクト `audio/03.m4a` に配置
6. 家来が `CONFIG.AUDIO_PLAYLIST` を3曲構成に拡張、push
