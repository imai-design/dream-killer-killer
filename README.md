# DREAM KILLER KILLER

夢を壊す奴を、壊す。¥10,000 を預けた100人だけのKILLER。

**LIVE:** https://imai-design.github.io/dream-killer-killer/

## Features
- 入場ゲート（タップ即BGM自動再生・ブラウザ規制クリア）
- 右上巨大サウンドトグル
- LIVE COUNTERS (4指標・counterapi.dev連携)
- THE 100 進捗バー＋6マイルストーン(1/10/20/50/70/100)
- マイルストーン到達で紙吹雪＋祝祭トースト
- BGM 2曲ローテート (AAC 128kbps)
- カスタムカーソル / グレイン / GSAP ScrollTrigger / Canvas particles

## Edit Points
`index.html` 冒頭 `CONFIG` ブロックを参照。
- `CURRENT_KILLERS`: 現在の確定KILLER数（手動更新）
- `PLEDGE_URL`: 決済リンク
- `FORM_ENDPOINT`: 申込POST先
- `AUDIO_PLAYLIST`: BGMファイル
